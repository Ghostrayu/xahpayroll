## [0.9.0] - 2024-12-15

INITIAL PUBLIC RELEASE OF XAH PAYROLL - DECENTRALIZED HOURLY PAYROLL SYSTEM ON XAHAU.

### ADDED

#### CORE FEATURES
- DECENTRALIZED HOURLY PAYROLL SYSTEM USING XAHAU PAYMENT CHANNELS
- HOURLY WAGE TRACKING WITH SINGLE PAYMENT SETTLEMENT UPON CHANNEL CLOSURE
- REAL-TIME WORK HOUR TRACKING AND SESSION MANAGEMENT
- XAMAN (XUMM) WALLET INTEGRATION WITH QR CODE SCANNING AND DEEP LINKING
- ROLE-BASED ACCESS CONTROL (EMPLOYEE VS NGO/EMPLOYER)
- WORKER MANAGEMENT WITH QR CODE SCANNING FOR EASY ONBOARDING

#### PAYMENT CHANNEL SYSTEM
- TWO-FIELD BALANCE ARCHITECTURE (OFF-CHAIN ACCUMULATED + ON-CHAIN ESCROW)
- PAYMENT CHANNEL CREATION WITH PRE-FLIGHT WALLET VALIDATION
- REAL-TIME CHANNEL ID RETRIEVAL VIA 3-TIER FALLBACK SYSTEM
- CHANNEL CLOSURE WITH SETTLEDELAY PROTECTION (24-72 HOURS)
- EXPIRED CHANNEL FINALIZATION WITH VISUAL STATUS INDICATORS
- AUTOMATIC ESCROW RETURN ON CHANNEL CLOSURE

#### WALLET INTEGRATION
- XAMAN WALLET EXCLUSIVE INTEGRATION WITH QR CODE SCANNING AND DEEP LINKING
- WALLET ADDRESS VALIDATION AND XAHAU VERIFICATION
- ACCOUNT ACTIVATION REQUIREMENT ENFORCEMENT (MINIMUM 10-20 XAH)
- MULTI-SIGNATURE TRANSACTION SUPPORT
- TESTNET AND MAINNET NETWORK SWITCHING
- ENTERPRISE-GRADE SECURITY WITH XAMAN'S KEY MANAGEMENT

#### WORKER MANAGEMENT
- ADD WORKERS TO ORGANIZATION VIA DASHBOARD
- SCAN WORKER WALLET ADDRESS WITH XAMAN QR CODE
- MULTI-ORGANIZATION WORKER SUPPORT (SAME WALLET, MULTIPLE EMPLOYERS)
- WORKER PAYMENT CHANNEL LISTING AND MANAGEMENT
- WORKER PROFILE DELETION WITH 48-HOUR GRACE PERIOD

#### DASHBOARD & UI
- NGO/EMPLOYER DASHBOARD WITH REAL-TIME CHANNEL MONITORING
- WORKER DASHBOARD WITH ACTIVE PAYMENT CHANNELS
- PAYMENT CHANNEL STATUS INDICATORS (ACTIVE, CLOSING, EXPIRED, CLOSED)
- ESCROW BALANCE TRACKING WITH NETWORK-AWARE LEDGER VERIFICATION
- ACTIVITY LOGS AND PAYMENT HISTORY
- RESPONSIVE DESIGN WITH TAILWINDCSS

#### BACKEND INFRASTRUCTURE
- POSTGRESQL DATABASE WITH COMPREHENSIVE SCHEMA
- EXPRESS.JS API WITH SECURITY MIDDLEWARE (HELMET, CORS, RATE LIMITING)
- JWT AUTHENTICATION WITH SESSION MANAGEMENT
- XRPL SDK INTEGRATION FOR LEDGER OPERATIONS
- DATABASE BACKUP AND RESTORE SCRIPTS
- ENVIRONMENT CONFIGURATION VALIDATION

#### SECURITY
- HELMET.JS SECURITY HEADERS
- CORS WITH CONFIGURABLE ORIGINS
- RATE LIMITING (100 REQUESTS PER 15 MINUTES)
- JWT TOKEN EXPIRATION (7 DAYS DEFAULT)
- DATABASE CREDENTIAL ENCRYPTION
- INPUT VALIDATION WITH JOI

#### DOCUMENTATION
- COMPREHENSIVE README.MD WITH SETUP INSTRUCTIONS
- CLAUDE.MD FOR AI-ASSISTED DEVELOPMENT GUIDANCE
- DATABASE SETUP GUIDE IN DOCUMENTS/
- PAYMENT CHANNEL TESTING GUIDE
- NETWORK CONFIGURATION DOCUMENTATION
- WALLET INTEGRATION GUIDES

### CHANGED
- REFACTORED DOCUMENTATION INTO DOCUMENTS/ DIRECTORY
- MIGRATED TO OFF-CHAIN ACCUMULATED BALANCE TRACKING FOR PERFORMANCE
- ENHANCED PAYMENT CHANNEL UI WITH STATUS INDICATORS
- IMPROVED ERROR HANDLING WITH CLEAR USER MESSAGES
- OPTIMIZED DATABASE QUERIES FOR PAYMENT CHANNEL RETRIEVAL

### FIXED
- PAYMENT CHANNEL CREATION FOR NON-ACTIVATED WALLETS (TECNO_DST ERROR)
- CHANNEL ID RETRIEVAL FALLBACK SYSTEM TO PREVENT TEMP IDS
- NGO DASHBOARD ESCROW BALANCE DISPLAY FOR CLOSING CHANNELS
- WORK SESSION NOTIFICATION HANDLING
- ENVIRONMENT CONFIGURATION VALIDATION

### SECURITY
